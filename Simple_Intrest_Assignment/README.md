- Excel Formulas

## Project Description
This Excel project analyzes simple interest calculations for multiple loan scenarios using principal amounts, interest rates, loan duration, total loan repayment calculations, and monthly payment estimations.

The assignment focuses on financial analysis techniques and loan repayment calculations using structured Excel formulas and chart visualization.
